
steps
-------

open command prompt, then go to the project directory and type following commands sequentially.

1. "npm -g install typescript" [-g indicates installing typescript globally]
	or
   "npm install typescript" for normal install.

NOTE: The above command creates node modules, TYPINGS and put them in saperate folder in your project automatically.



2. npm -g install typings.
3. npm start


NODE MODULES gets created when you run the project.
